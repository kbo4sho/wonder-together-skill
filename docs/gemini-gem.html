Open the Wonder Together Gemini Gem.

Gemini Gems are the closest Gemini equivalent to a Custom GPT: a reusable custom assistant with a name, description, instructions, and starters. Use the live Wonder Together Gem, or recreate it from the public instructions below if you maintain a separate copy.

Name

Wonder Together

Use this as the Gem name.

Description

Short and clear

Generate one warm hand-drawn Pebble Guides image from a child's question.

Default tool

Use Create image.

Set the Gem default tool to Images / Create image. Wonder Together prioritizes the inline generated image. The Gem instructions are intentionally short and image-tool focused: every parent message becomes a cream-paper Pebble Guides visual recipe before Gemini generates the image.

Instructions

Paste the public Gem instructions.

Copy the `gemini_gem.instructions` block from the repo source. Share only this public instruction material; do not upload private examples, child data, or anything that should not be visible to people who can access the Gem.

Before sharing

Test the Gem with three kinds of question.

The Gem should generate one inline image first. If Gemini cannot generate an image in the chat, it should provide one polished image prompt instead of blocking the parent.

Ordinary science

My child is 6 and asked why the moon follows our car.

Hazard

My child is 7 and asked where the electricity in the wall comes from.

Tender topic

My child is 5 and asked if I will die someday.

Expected behavior

Make the image first, keep the Pebble Guides visible, avoid unsafe activities, use direct gentle wording for death, and keep fuller stories or games as optional follow-ups.
